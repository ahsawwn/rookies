## Image Selection Tips

1. **Choose square images** (1:1 aspect ratio) for best display
2. **Clean backgrounds** - white or neutral backgrounds work best
3. **Good lighting** - well-lit products look more appetizing
4. **High resolution** - at least 800x800px, preferably 1200x1200px or larger
5. **Professional quality** - avoid blurry or low-quality images
6. **Accurate representation** - image should match the product description

## Download Instructions

1. Visit one of the recommended websites
2. Search for the product name
3. Filter by "Free" and "High Resolution"
4. Download the image
5. Save it in the appropriate category folder with the correct filename
6. Ensure the image is in JPG or PNG format

## File Naming Convention

Use lowercase with hyphens, matching the product name:
- `nankhatai.jpg`
- `chocolate-chip-cookies.jpg`
- `mango-shake.jpg`
- `vanilla-cupcake.jpg`
- `chocolate-cake-1kg.jpg`
